SerializerRegisterType 方法

向 Serializer 注册类型与主键集合。

命名空间：  XPatchLib
程序集：  XPatchLib (在 XPatchLib.dll 中) 版本：4.0.2.3
语法
C#
复制
public void RegisterType(
	Type pType,
	string[] pPrimaryKeys
)

参数

pType
类型： SystemType
待注册的类型。
pPrimaryKeys
类型： SystemString
pType 的主键名称集合。
一场
异常条件
ArgumentNullException 当参数 pType 或 pPrimaryKeys is null 时。
ArgumentException 当参数 pPrimaryKeys.Length == 0 时。
备注
在无法修改类型定义，为其增加或修改 PrimaryKeyAttribute 的情况下， 可以在调用 Divide 或 Combine 方法前，调用此方法，传入需要修改的Type及与其对应的主键名称集合。 系统在处理时会按照传入的设置进行处理。
