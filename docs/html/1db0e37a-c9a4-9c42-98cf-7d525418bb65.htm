ITextWriterIndentation 属性

获取或设置当 Formatting 设置为 Formatting.Indented 时将为层次结构中的每个级别书写多少 IndentChar。

命名空间：  XPatchLib
程序集：  XPatchLib (在 XPatchLib.dll 中) 版本：4.1.0.4
语法
C#
复制
int Indentation { get; set; }

属性值

类型： Int32
每个级别的 IndentChar 的数目。默认值为 2。
