ITextReaderMoveToNextElement 方法

从流中读取下一个同级节点。

命名空间：  XPatchLib
程序集：  XPatchLib (在 XPatchLib.dll 中) 版本：3.0.0.1
语法
C#
复制
bool MoveToNextElement(
	string nextElementName,
	string parentElementName
)

参数

nextElementName
类型： SystemString
下一个同级节点的名称。
parentElementName
类型： SystemString
父级节点的名称。

返回值

类型： Boolean
如果成功读取了下一个节点，则为 true；如果没有其他节点可读取，则为 false。
